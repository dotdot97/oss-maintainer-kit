# AGENTS.md

## Project goal

This project helps practice open-source maintainer workflows such as documentation review, issue triage, pull request review, release note drafting, and responsible AI-assisted maintenance.

## Commands

- Run tests: `pytest`
- Run module: `python -m oss_maintainer_kit`

## Rules for AI-assisted changes

- Do not invent project adoption, users, stars, downloads, or maintainer status.
- Keep changes small and reviewable.
- Add or update tests for behavior changes.
- Update README when user-facing behavior changes.
- Do not auto-merge generated changes.
- Keep explanations beginner-friendly.
