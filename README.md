# OSS Maintainer Kit

A lightweight starter toolkit for learning and documenting open-source maintainer workflows.

> Status: early-stage learning project. This repository is being built to practice documentation, issue triage, pull request review, release notes, and basic automation.

## Why this project exists

Open-source maintainers repeatedly handle documentation checks, issue triage, pull request review, release notes, and contributor onboarding. This project starts as a small, honest learning repository for building those workflows step by step.

## Current features

- Basic command-line entry point
- README quality checklist
- Issue triage checklist
- Pull request review checklist
- Release notes template
- GitHub issue and PR templates
- Basic Python test setup

## Planned features

- README structure checker
- Markdown link checker
- Issue label suggestion helper
- Changelog draft generator
- Pull request template validator

## Usage

This project is currently a starter repository. The Python CLI is intentionally minimal.

```bash
python -m oss_maintainer_kit
```

Expected output:

```text
OSS Maintainer Kit: ready
```
