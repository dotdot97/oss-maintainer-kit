# Code of Conduct

## Our pledge

We aim to maintain a respectful and beginner-friendly open-source learning environment.

## Expected behavior

- Be respectful and constructive.
- Explain feedback clearly.
- Help beginners understand the reason behind changes.
- Avoid harassment, insults, and personal attacks.

## Enforcement

The maintainer may close issues, reject pull requests, or restrict participation if behavior is disruptive.
